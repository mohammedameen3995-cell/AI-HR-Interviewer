🤖 AI HR Interviewer
📌 Overview

AI HR Interviewer is an intelligent virtual interview system that automates the hiring process using AI. It analyzes resumes, conducts real-time interviews, evaluates candidate responses, and provides detailed performance insights to help recruiters make smarter decisions.

🚀 Key Features
🎤 Automated Interviews – AI-driven question generation
📄 Resume Analysis – Extracts skills, experience, and insights
🧠 Smart Evaluation – Analyzes answers using NLP
📊 Performance Scoring – Provides detailed feedback & ratings
💬 Dynamic Questions – Adapts based on candidate responses
🌐 Multi-domain Support – IT, Business, HR, etc.
🗣️ Voice/Text Interaction – Interactive interview experience
